# 🛡️ RouterWatch - حارس شبكتك المنزلية
**التحكم الكامل في وقت الإنترنت.. بين يديك.**

روتر ووتش (RouterWatch) هو الحل الأمثل لكل أسرة تسعى لتنظيم وقت استخدام الإنترنت للأبناء وحمايتهم من الإفراط الرقمي. يمنحك البرنامج القدرة على إدارة جهاز الراوتر الخاص بك بواجهة عربية بسيطة وسهلة، دون الحاجة للدخول في إعدادات الراوتر المعقدة.

## ✨ أبرز المميزات
* **🕒 تنظيم الوقت:** حدد ساعات العمل وساعات النوم لكل جهاز (مثلاً: إغلاق الإنترنت تلقائياً عن أجهزة الألعاب وقت المذاكرة أو النوم).
* **📱 كشف الأجهزة اللحظي:** رؤية واضحة لجميع الهواتف، الحواسيب، وأجهزة الألعاب المتصلة بشبكتك الآن.
* **🔒 خصوصية تامة:** جميع بياناتك وكلمات مرورك تُحفظ بشكل مشفر على جهازك الشخصي فقط، ولا يتم مشاركتها أبداً.
* **🚀 سهولة الاستخدام:** بنقرة واحدة يمكنك تفعيل أو إلغاء قواعد الحظر دون الحاجة لمعرفة تقنية مسبقة.
* **🔄 تحديثات مستمرة:** البرنامج يطور نفسه تلقائياً لضمان التوافق الدائم مع أجهزة الراوتر المنزلية الحديثة.
